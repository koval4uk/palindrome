Задание:
Написать приложение, которое будет находить самый длинный палиндром в файле с текстом.
Пользователь должен иметь возможность задать через консоль:

файл с данными
файл для вывода результата

Пример команды:

java -jar Palindrom.jar "input.txt" "result.txt"

Пример команды c дополнительной сложностью:

java -jar Palindrom.jar "input.txt" "result.txt" [ parallel ] [ line ]

Условия

реализация в одном потоке
проверка палиндромов по словам
Если в файле встречается несколько палиндромов максимальной длинны, то вывести все палиндромы в новых строках в порядке натуральной сортировки (ASC)
не забудьте упаковать решение в JAR (Palindrom.jar) и добавить в корень репозитория


Дополнительная сложность (не обязательно, но желательно)

Выбор типа поиска:
** добавить поиск палиндромов в тексте по строкам (палиндром может вмещать whitespaces внутри (по бокам только видимые символы). Пример: алгоритм нашел " а bb а ", тогда в результате будет "а bb а"). Параметр "line"
метод исполнения поиска палиндромов:
** используя многопоточность. Параметр "parallel"
